# 지원 대상 및 입과 안내

## 지원 대상

CodeAI 부트캠프는 다음에 해당하는 분들을 위한 과정입니다.

### ✅ 적합한 지원자

- **비전공자 직장인**: IT 비전공이지만 AI 분야로 커리어 전환을 희망하는 분
- **취업 준비생**: AI/개발 분야 첫 취업을 목표로 하는 분 (전공 무관)
- **주니어 개발자**: 개발 경험은 있지만 AI 역량을 추가하고 싶은 분
- **창업 준비자**: AI 기반 서비스나 스타트업을 준비 중인 분
- **국민내일배움카드 소지자**: 정부 지원을 통해 실비로 교육받고 싶은 분

### 📌 권장 사전 지식

| 수준 | 내용 |
|------|------|
| **필수** | 컴퓨터 기본 조작 능력 (인터넷 검색, 파일 관리) |
| **권장** | Python 기초 문법 경험 (변수, 함수, 조건문 수준이면 충분) |
| **우대** | HTML/CSS 기초, 영어 기술 문서 독해 가능 |

> ※ Python을 전혀 모르셔도 과정 초반 Python 특강(3주)이 제공되므로 걱정하지 않으셔도 됩니다.

### ❌ 지원이 어려운 경우

- 고등학생 이하 (만 19세 미만)
- 재학 중인 대학생 (단, 졸업 예정자 예외 적용 가능)
- 이미 동일 또는 유사 훈련과정을 수료한 후 6개월이 경과하지 않은 분

## 입과 절차

```
온라인 신청서 제출
        ↓
서류 검토 (3~5 영업일)
        ↓
전화/화상 상담 인터뷰 (약 15분)
        ↓
합격 통보 및 입과 안내 이메일 발송
        ↓
등록금 납부 (합격 통보 후 7일 이내)
        ↓
OT 참석 후 과정 시작
```

## 신청 방법

1. **온라인 신청**: CodeAI 공식 홈페이지 → 신청하기 버튼 클릭
2. **신청 링크**: [https://apply.codeai.kr](https://apply.codeai.kr)
3. **신청 마감**: 2026년 6월 20일 (금) 또는 정원 마감 시 조기 종료

## 제출 서류

- 신청서 (온라인 작성)
- 최종 학력 증명서 1부
- 국민내일배움카드 신청자: 카드 사진 또는 카드번호
