# 커리큘럼

CodeAI 부트캠프는 총 6개월(약 1,000시간) 과정으로, 4개의 단계(Phase)로 구성됩니다.

## 전체 커리큘럼 개요

| Phase | 기간 | 주제 | 기간 |
|-------|------|------|------|
| Phase 1 | 1~4주차 | Python & 개발 환경 기초 | 4주 |
| Phase 2 | 5~12주차 | AI/ML 핵심 이론 & 실습 | 8주 |
| Phase 3 | 13~20주차 | LLM & AI 서비스 개발 | 8주 |
| Phase 4 | 21~26주차 | 팀 프로젝트 & 취업 준비 | 6주 |

---

## Phase 1: Python & 개발 환경 기초 (1~4주차)

### 1주차: 개발 환경 구축 & Python 기초
- Git & GitHub 사용법
- VS Code 설정 및 확장 프로그램
- Python 설치 및 가상환경 관리 (venv, conda)
- Python 기초 문법: 변수, 자료형, 조건문, 반복문

### 2주차: Python 심화
- 함수, 클래스, 모듈
- 파일 입출력, 예외 처리
- 라이브러리 활용 (os, json, csv)
- 미니 프로젝트: 데이터 파싱 & 처리 스크립트 작성

### 3주차: 데이터 분석 기초
- NumPy 핵심 연산
- Pandas 데이터프레임 다루기
- Matplotlib / Seaborn 시각화
- 실습: 공공데이터 분석 및 시각화

### 4주차: 웹 기초 & API 이해
- HTTP, REST API 개념
- FastAPI 입문
- Requests 라이브러리로 API 호출
- 미니 프로젝트: 날씨 API 연동 CLI 앱

---

## Phase 2: AI/ML 핵심 이론 & 실습 (5~12주차)

### 5~6주차: 머신러닝 기초
- 지도학습 / 비지도학습 개념
- Scikit-learn: 분류, 회귀, 클러스터링
- 데이터 전처리 (결측값, 정규화, 인코딩)
- 모델 평가 (정확도, F1-Score, 혼동행렬)

### 7~8주차: 딥러닝 기초
- 신경망 구조 이해 (퍼셉트론 → MLP)
- PyTorch 기초 문법
- 이미지 분류 CNN 구현
- 미니 프로젝트: 손글씨 이미지 분류기

### 9~10주차: 자연어 처리 (NLP)
- 텍스트 전처리 (토크나이징, 불용어)
- Word Embedding (Word2Vec, GloVe)
- Transformer 아키텍처 이해
- BERT 파인튜닝 실습 (감정 분석)

### 11~12주차: AI 모델 서빙
- FastAPI로 ML 모델 API 서버 구축
- Docker 기초 및 컨테이너화
- 클라우드 배포 (AWS EC2 / Railway)
- 미니 프로젝트: 감정 분석 API 서버 배포

---

## Phase 3: LLM & AI 서비스 개발 (13~20주차)

### 13~14주차: LLM 기초 & OpenAI API
- LLM(대형 언어 모델) 개념 이해
- OpenAI API 사용법 (Chat, Embedding)
- Prompt Engineering 기법
- 실습: GPT 기반 텍스트 요약/분류 앱

### 15~16주차: RAG (검색 증강 생성)
- RAG 아키텍처 이해
- 문서 청킹 및 벡터 임베딩
- FAISS / Chroma 벡터 DB 사용법
- 미니 프로젝트: 사내 문서 기반 Q&A 챗봇

### 17~18주차: AI 에이전트 & 도구 연동
- LangChain 프레임워크 활용
- Tool Calling / Function Calling
- 웹 검색, 코드 실행 에이전트 구현
- 실습: 자율 검색 리서치 에이전트

### 19~20주차: 풀스택 AI 서비스 개발
- React + TypeScript 프론트엔드
- TailwindCSS UI 구성
- FastAPI + React 통합
- 미니 프로젝트: AI 상담 챗봇 풀스택 구현

---

## Phase 4: 팀 프로젝트 & 취업 준비 (21~26주차)

### 21~23주차: 팀 프로젝트 (기획 & 개발)
- 팀 구성 (3~4인) 및 주제 선정
- 기획서 작성 및 발표
- 스프린트 방식 개발 진행
- 강사 멘토링 (주 2회)

### 24~25주차: 팀 프로젝트 (마무리 & 발표)
- 프로젝트 완성 및 배포
- 발표 자료 제작
- 데모 데이 (기업 인사담당자 초청)
- 포트폴리오 GitHub 정리

### 26주차: 취업 준비 & 수료
- 이력서 / 포트폴리오 최종 코칭
- 모의 기술 면접 (1:1)
- 협력 기업 채용 연계
- 수료식 및 수료증 발급
