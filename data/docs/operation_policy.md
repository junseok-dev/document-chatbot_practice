# 운영 정책

## 교육 시간 및 일정

### 기본 교육 시간

| 구분 | 시간 |
|------|------|
| 평일 (월~금) | 오전 09:00 ~ 오후 06:00 |
| 점심 시간 | 낮 12:00 ~ 오후 01:00 (1시간) |
| 실제 교육 시간 | 하루 8시간 |

### 공휴일 및 방학

- 법정 공휴일에는 교육을 진행하지 않습니다.
- 추석 연휴 및 설 연휴는 별도 휴강 공지에 따릅니다.
- 별도 단기 방학은 없으며, 과정 중 공휴일이 포함된 주에는 해당일만 휴강합니다.

## 출결 관리

### 출석 기준

- 매일 오전 9시 정각에 출석 체크를 진행합니다.
- 출결은 **교육 관리 시스템(LMS)**에 자동 기록됩니다.

### 지각 / 결석 기준

| 구분 | 기준 |
|------|------|
| 정상 출석 | 09:00까지 입실 |
| 지각 | 09:01 ~ 09:30 사이 입실 |
| 조기 퇴실 | 16:00 이전 퇴실 |
| 결석 | 미입실 또는 지각 3회 = 결석 1회 |

### 출석률 기준

- **국민내일배움카드 훈련생**: 전체 과정의 **80% 이상** 출석 필수
  - 출석률 미달 시 훈련비 환수 조치 및 과정 수료 불인정
- **일반 수강생**: 70% 이상 출석 권장 (수료증 발급 기준)

### 결석 사유 인정 범위

아래 사유에 해당하는 경우 공결(공식 결석)로 처리되어 출석률에 불이익이 없습니다.

- 본인 또는 직계가족 경조사 (결혼, 장례 등) — 최대 3일
- 의사 진단서 제출 시 병가 — 최대 5일
- 천재지변, 자연재해 등 불가피한 사유

> 공결 처리는 반드시 **사전 또는 당일 오전 중** 담당자에게 연락해야 합니다.

## 과제 및 평가

### 과제 제출

- 매 Phase 종료 시 **미니 프로젝트 과제**가 부여됩니다.
- 과제는 GitHub 저장소에 제출하는 방식으로 진행합니다.
- 제출 기한: 각 Phase 종료 후 3일 이내

### 평가 방식

| 항목 | 비중 |
|------|------|
| 출석률 | 20% |
| 미니 프로젝트 (Phase 1~3) | 45% |
| 팀 최종 프로젝트 (Phase 4) | 35% |

### 수료 기준

- 출석률 80% 이상
- 전체 평가 점수 60점 이상
- 위 두 가지를 모두 충족해야 수료증이 발급됩니다.

## 행동 강령 및 준수 사항

### 교육장 이용 규칙

- 교육장 내 음식물 섭취는 지정 공간(라운지)에서만 가능합니다.
- 강의 중 타인에게 방해가 되는 행동(큰 소리, 이석 등)은 자제해 주세요.
- 개인 장비(노트북)를 사용하는 것을 원칙으로 하며, 노트북이 없는 경우 사전 안내 후 대여 가능합니다.

### 금지 행위

- 강의 내용 무단 녹화 · 촬영 · 배포
- 교육 자료 외부 유출
- 타인의 코드 무단 복사 제출 (학습 부정행위)
- 위 행위 적발 시 즉시 퇴소 처리됩니다.

## 노트북 및 장비 안내

### 권장 사양

| 항목 | 권장 사양 |
|------|----------|
| OS | Windows 11 / macOS 13 이상 |
| CPU | Intel i5 10세대 이상 또는 M1 이상 |
| RAM | 16GB 이상 |
| 저장공간 | SSD 256GB 이상 |

> 최소 RAM 8GB는 되어야 실습이 가능합니다. RAM 8GB 미만 장비는 교육 진행이 어려울 수 있습니다.

### 대여 장비

- 교육 기간 중 노트북 대여가 필요한 경우 입과 전 담당자에게 사전 신청 가능합니다.
- 대여 비용은 무료이며, 수료 후 반납해야 합니다.
