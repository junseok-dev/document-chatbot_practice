# 취업 지원 프로그램

## 개요

CodeAI 부트캠프는 수료 후 **6개월간** 취업 지원 서비스를 무상으로 제공합니다.
단순한 구인·구직 연결을 넘어, 실질적인 취업 경쟁력을 높이는 것을 목표로 합니다.

---

## 취업 지원 항목

### 1. 이력서 & 포트폴리오 코칭

- **이력서 코칭**: 현직 개발자 출신 강사가 1:1로 이력서를 검토하고 피드백 제공
- **포트폴리오 정리**: GitHub 프로필, README 작성, 프로젝트 설명 문서화 지원
- **코칭 횟수**: 1인당 최대 3회 무상 제공 (수료 후 6개월 이내)

### 2. 모의 기술 면접

- **대상**: 수료 후 취업 준비 중인 수강생
- **방식**: 1:1 화상 모의 면접 (약 40분)
- **평가 항목**: Python 기초, AI/ML 개념, 프로젝트 설명, 코딩 테스트 기초
- **횟수**: 1인당 최대 2회 무상 제공
- **피드백**: 면접 후 서면 피드백 제공

### 3. 협력 기업 채용 연계

현재 CodeAI와 채용 협약을 맺은 협력 기업 목록입니다.

| 분야 | 협력 기업 (예시) |
|------|---------------|
| AI 스타트업 | AI 솔루션 스타트업 8개사 |
| IT 서비스 | 중견 IT 서비스 기업 5개사 |
| 금융 IT | 핀테크 기업 3개사 |
| 교육 플랫폼 | EdTech 기업 4개사 |

> ※ 정확한 협력 기업 목록은 수료 후 취업 지원 오리엔테이션에서 공개됩니다.

### 4. 취업 준비 스터디

- **AI 코딩 테스트 스터디**: 주 2회 온라인 (수료 후 2개월간)
- **알고리즘 기초 스터디**: 주 1회 온라인 (선택 참여)
- **현직자 커리어 토크**: 월 1회 협력 기업 현직자 특강

### 5. 수료생 커뮤니티 (Alumni)

- **CodeAI Alumni Slack**: 수료생 전용 채널 영구 가입
- **정기 네트워킹**: 분기별 1회 오프라인 모임
- **잡 레퍼럴 채널**: 수료생 간 채용 정보 공유 채널
