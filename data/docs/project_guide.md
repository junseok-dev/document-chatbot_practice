# 최종 팀 프로젝트 안내

## 프로젝트 개요

Phase 4에서 진행하는 **팀 프로젝트**는 CodeAI 부트캠프의 핵심 산출물입니다.
3~4인 팀을 구성하여 실제 서비스 수준의 AI 기반 웹 애플리케이션을 기획, 개발, 배포하고,
데모 데이에서 기업 인사담당자 앞에서 발표합니다.

---

## 팀 구성

- **인원**: 팀당 3~4명
- **구성 방식**: 강사 배정 방식 (역할 균형 고려)
- **역할 분담 예시**:
  - PM(기획 담당) 1명
  - AI/백엔드 개발 1~2명
  - 프론트엔드 개발 1명

---

## 프로젝트 주제 조건

팀 프로젝트 주제는 다음 조건을 반드시 충족해야 합니다.

### 필수 조건

- [x] AI 기능이 핵심 역할을 담당해야 함
- [x] LLM API 또는 ML 모델 중 하나 이상 사용
- [x] FastAPI 백엔드 + React 프론트엔드 구조 (풀스택)
- [x] 실제 배포 완료 (외부 접속 가능한 URL 존재)
- [x] GitHub 저장소 공개 및 README 완성

### 주제 예시 (참고용)

| 분야 | 예시 주제 |
|------|----------|
| 교육 | AI 맞춤형 학습 플랜 생성기 |
| 헬스케어 | 식단 이미지 분석 & 영양 상담 챗봇 |
| 비즈니스 | 사내 문서 기반 Q&A 챗봇 |
| 생산성 | AI 회의록 자동 요약 & 액션 아이템 추출 |
| 커머스 | 리뷰 감정 분석 & 상품 추천 시스템 |
| 크리에이티브 | AI 소설/시나리오 공동 창작 플랫폼 |

> 위 예시 외에도 자유롭게 주제를 제안할 수 있으며, 강사 승인 후 진행합니다.

## 제출물 목록

최종 제출 시 아래 항목이 모두 완료되어야 합니다.

1. **GitHub 저장소** (공개)
   - `README.md`: 프로젝트 소개, 기술 스택, 실행 방법, 팀원 소개
   - 소스 코드 전체
   
2. **배포 URL**: 외부에서 접속 가능한 서비스 주소

3. **발표 자료** (PPT 또는 Notion): 20슬라이드 내외

4. **시연 영상** (선택): 3분 내외 데모 영상 (YouTube 또는 Google Drive)
