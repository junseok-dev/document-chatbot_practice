## 핵심 가치

1. **데이터 엔지니어링 통합**
   - 데이터 기본기부터 MLOps 자동화까지 학습.
   - 순차 학습: Python·SQL·NoSQL → Spark·Kafka → Docker·K8s → MLflow·Airflow.

2. **교과**
   - 엔코아의 데이터 컨설팅 경험 기반 레거시 데이터를 학습용으로 변환하는 실습.

3. **자동화 완성**
   - 전체 자동화: 모델 생애주기 전체 자동화, 장애 자동 복구, 무중단 배포, 지속 학습 시스템 직접 구축.
