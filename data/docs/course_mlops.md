## 핵심 가치

1. **데이터 엔지니어링 통합**
   - 데이터 기본기부터 MLOps 자동화까지 학습
   - Python, SQL, NoSQL → Spark, Kafka → Docker, K8s → MLflow, Airflow 순차 학습

2. **교과**
   - 데이터 컨설팅 경험 기반 레거시 데이터를 학습용으로 변환하는 노하우 실습

3. **자동화 완성**
   - 모델 생애주기 전체 자동화
   - 장애 자동 복구, 무중단 배포, 지속 학습 시스템 직접 구축

## 추천 대상
- 데이터 인프라 시스템 운영에 관심 있는 분
- 비전공자도 Vibe Coding으로 진입 가능
- 신입 및 1~2년차 실무 투입 가능 역량 양성

## 과정 요약
- **왜 지금 MLOps 엔지니어인가**
   - 모델 만들기는 하나 운영이 안 되는 국내 대기업의 공통된 공백
   - 데이터 파이프라인, 모델 운영 자동화 통합 역량의 수요 증가
   - MLOps의 데이터 엔지니어링 확장이 2026년 표준으로 정착 중

## 수료 시 확보하는 결과물 카드

- **Lv.1 (STEP 1, 2)**: 데이터 설계자
    - SQL, NoSQL 파이썬 기반 데이터 파이프라인 구축 프로젝트
    - 시장 가치 데이터 엔지니어링 실무에 활용 가능한 주니어 수준 포트폴리오

- **Lv.2 (STEP 3)**: 대용량 데이터 전문가
    - Spark, Kafka 기반 분산 처리 시스템 구축
    - 데이터 플랫폼 엔지니어 수준 포트폴리오

- **Lv.3 (STEP 4, 5)**: 모델 운영 시스템 구축자
    - MLflow, Airflow, Docker, K8s 기반 모델 생애주기 자동화
    - MLOps 엔지니어 수준 포트폴리오
